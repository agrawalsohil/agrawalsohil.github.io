# Portfolio Site — Sohil Agrawal

Personal portfolio website for **Sohil Agrawal**, Data &amp; AI Solution Architect.

- **Single file**, zero build: everything (HTML + CSS) lives in [`index.html`](index.html).
- **Self-contained**: no external scripts, fonts, or image requests — renders identically offline and behind corporate proxies (e.g. Zscaler).
- **Responsive** and supports automatic **light / dark mode** (`prefers-color-scheme`).

## Run locally
Just open `index.html` in any browser. No server or dependencies needed.
